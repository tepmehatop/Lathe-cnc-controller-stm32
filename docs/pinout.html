🔌 DRO HS800-2

Протокол: UART 57600/8N1
Пакет: 29 байт, CRC16/MODBUS
TX → PA3 (USART2 RX)
GND → GND
Уровни: 3.3В (совместим)

🖥 LCD2004 (старый дисплей)

Протокол: I2C @ 400кГц
Адрес: 0x27 (PCF8574)
SDA → PB7 (I2C1)
SCL → PB6 (I2C1)
VCC: 5В, GND: GND

🔄 Энкодер шпинделя

1800 линий/оборот
Ch A → PA0 (TIM5_CH1 / EXTI0)
Ch B → PA1 (TIM5_CH2 / EXTI1)
VCC: 5В, GND: GND
Режим: счётчик таймера или EXTI

🎛 Ручной энкодер (100 линий)

Ch A → PC2 (EXTI2)
Ch B → PC3 (EXTI3)
Ось Z/X → PD0, PD1
Шаг ×1/×10 → PD2, PD3
Подтяжка: внутренняя PULLUP

🕹 Джойстик + Rapid

Влево → PC4
Вправо → PC5
Вверх → PC6
Вниз → PC7
Rapid → PC8
Активный уровень: LOW (подтяжка PULLUP)

STM32F407VGT6

STM32F4DISCOVERY
🔧 168 МГц Cortex-M4F
💾 1 MB Flash / 192 KB RAM
⚡ FPU (float без задержек)
📡 6× UART + DMA
🔗 2× I2C
⏱ 14× 16/32-bit Timers
🔌 2× CAN (запас)
📺 USB OTG FS/HS

PA0 — USER BTN ⚠
PD12-15 — LEDs ⚠
PB3 — SWO debug
PA13/PA14 — SWD
⚠ Пины могут быть переназначены
под разводку шилда

📺 ESP32-S3 JC4827W543 (новый дисплей)

Протокол: UART 115200/8N1
TX → PE0 (UART8 TX)
RX ← PE1 (UART8 RX)
ESP32 GPIO44 (RX) ← PE0
ESP32 GPIO43 (TX) → PE1
DMA: обновление без блокировки CPU

⚙ Мотор Y (продольная)

Драйвер: шаговый (Step/Dir/En)
STEP → PE9 (TIM1_CH1)
DIR → PE10
EN → PE11
Таймер TIM1, DMA для импульсов

⚙ Мотор X (поперечная)

Драйвер: шаговый (Step/Dir/En)
STEP → PE13 (TIM1_CH3)
DIR → PE14
EN → PE15
Таймер TIM1, общий с Y

🛑 Лимитные выключатели + LED

BTN Left → PF0 | LED Left → PF1
BTN Right → PF2 | LED Right → PF3
BTN Front → PF4 | LED Front → PF5
BTN Rear → PF6 | LED Rear → PF7
Кнопки: PULLUP, активный LOW
LED: открытый коллектор, 470Ом

🔘 Кнопки меню + Переключатели

BTN Left/Right/Up/Down → PG0-PG3
BTN Select → PG4
Submode SW (3 pos) → PG5, PG6, PG7
Mode SW (8 pos) → PG8-PG15
Tacho out → PE8
Beeper → PD12 (TIM4_CH1)
⚠ Примечание: Данная схема — предварительная. Пины будут уточнены при разработке шилда. Все назначения вынесены в Core/els_config.h — достаточно изменить один файл. | Питание 3.3В для сигналов. Для 5В-периферии (LCD2004, драйверы) — уровень питания через шилд.
